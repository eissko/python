### Install Ldap
```
sudo apt-get install libsasl2-dev python-dev libldap2-dev libssl-dev
```
```
pip3 install python-ldap
```

### better is to use ldap 3
- https://pypi.org/project/ldap3/

```
python -m pip install ldap3
```
